# Update a project

## Steps

1. **Find your project** in `data/ecosystem.ts` (search by name or UUID)

2. **Edit the fields** you want to update (description, tags, links, logo, etc.)

3. **If updating your logo**, replace the file in `public/logos/` with the same filename, or add a new file and update the `image` field

4. **Run the linter**:

```bash
yarn lint
```

5. **Create a Pull Request** with the title: `ðŸŽ¨ Updating YourProjectName`

Your changes will go live automatically once the PR is merged.
